# Natural Language Processing (NLP)

This folder contains work related to Natural Language Processing techniques and applications.

## Overview
This project contains a series of exercises, tutorials (TD/TPs), and a final comprehensive project focusing on text data analysis and NLP-based recommendation systems.

## Key Contents
- **Projet_NLP_Recommandation_FINAL.ipynb**: The final project notebook implementing an NLP-based recommendation engine.
- **TP/TDs**: Various practical sessions covering fundamental and advanced NLP concepts (RNN, LSTM, etc.).

## Technologies Used
- Python
- NLP Libraries (NLTK, Spacy, Transformers, etc.)
- Deep Learning (TensorFlow/Keras or PyTorch)
- Jupyter Notebooks
