# Prédiction de Pluie (Rain Prediction)

Ce projet d'apprentissage automatique a pour objectif de prédire les précipitations (pluie) en utilisant des données météorologiques historiques.

## Contenu du projet
- **TP2.ipynb** : Le notebook Jupyter contenant l'analyse exploratoire des données (EDA), le nettoyage, et l'entraînement des modèles de prédiction (Random Forest, Régression Logistique, etc.).
- **rapport_prediction_pluie_final.pdf** : Le rapport final détaillant la méthodologie, les expérimentations menées, l'évaluation des différents modèles et les conclusions de l'étude.

## Technologies utilisées
- Python
- Pandas & NumPy pour le traitement des données
- Scikit-Learn pour la modélisation prédictive
- Matplotlib & Seaborn pour la visualisation de données
